# Семейный Наставник

Локальный AI-наставник для детей 7–14 лет на базе **Gemma 4** (Google).

## Особенности
- Три уникальных наставника: Кибер, Солнечный, Мудрец (строгие промпты с правилами сессий)
- Родительский контроль: реальный лимит времени (разница между вопросами) + логи
- Полная локальность: данные не покидают ваш компьютер
- Адаптивный интерфейс (ПК, планшеты, телефоны)
- Лаунчер с пользовательским согласием и автоматической проверкой/установкой Ollama

## Требования
- Windows 10/11
- Python 3.10+ (добавьте в PATH)
- 8 ГБ ОЗУ (рекомендуется 16 ГБ)
- Свободное место ~6 ГБ
- Интернет для первоначальной загрузки модели (~9.6 ГБ)

## Установка и запуск

### 1. Установка Ollama
Скачайте и установите `OllamaSetup.exe` с [официального сайта](https://ollama.com/download/windows).

### 2. Загрузка модели
Откройте командную строку (администратор) и выполните:
